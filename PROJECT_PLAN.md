# خطة مشروع قطعتي (Qet3aty)

## الشعار
سوق الحرفيين تحت إيدك

## الهدف
إنشاء أكبر منصة لبيع وشراء قطع غيار السيارات في مصر.

## النسخة الأولى (MVP)

### المستخدم
- تسجيل الدخول
- إنشاء حساب
- البحث عن قطع الغيار
- عرض المنتجات
- إضافة للسلة
- المفضلة
- إتمام الطلب
- متابعة الطلب

### التاجر
- تسجيل متجر
- إضافة منتج
- تعديل الأسعار
- إدارة الطلبات
