# 🚗 Qet3aty - قطعتي

## سوق الحرفيين تحت إيدك

### نبذة عن المشروع
قطعتي هو تطبيق متخصص في بيع وشراء قطع غيار السيارات في مصر، يربط بين أصحاب السيارات والتجار والحرفيين في مكان واحد.

## أهداف المشروع
- البحث عن قطع الغيار بسهولة.
- مقارنة الأسعار بين التجار.
- الوصول لأقرب متجر.
- طلب القطعة أونلاين.
- دعم جميع ماركات السيارات.

## مميزات النسخة الأولى (MVP)
- تسجيل الدخول.
- إنشاء حساب.
- الصفحة الرئيسية.
- البحث.
- الأقسام.
- صفحة المنتج.
- السلة.
- المفضلة.
- حساب المستخدم.
